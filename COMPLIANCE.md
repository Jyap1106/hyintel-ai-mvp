# Compliance and Data Use

## MVP compliance position

HyIntel AI MVP uses approved uploaded/sample data only.

## Allowed

- Local mock data
- Approved sample candidate profiles
- Approved sample company data
- Approved sample salary data
- Simulated market intelligence
- Human-reviewed recommendations

## Not allowed

- Live scraping
- Real LinkedIn integration
- Real JobStreet integration
- Real Indeed integration
- External recruitment platform calls
- Automated hiring decisions
- Unsupported candidate claims
- Hidden scoring logic

## Required UI labels

- Approved sample data
- Evidence-based
- Human review required
- Demo intelligence only

## Required compliance copy

Compliance-first MVP: This demo uses approved uploaded/sample data only. Candidate recommendations are evidence-based and require human review. The system does not make automated hiring decisions.

## Meaning of "all companies"

When the UI says "all companies," it means:

All companies found in the approved demo dataset for the selected role.
